##问题##

因为我下载的是Markdownpad2工具，所以我就以这个工具遇到的问题阐述一下，如果你的系统是win10，在安装完Markdown还会发生如下图所示的问题。

这个时候可能你会很慌，怎么在刚下载的东西都不能用了，是不是安装步骤错误了，其实并不是，你只要下载这个东西awesomium_v1.6.6_sdk_win，然后重启一下MarkDown就可以了

如果还是不行的话就执行以下步骤：
下载Microsoft’s DirectX End-User Runtimes (June 2010)，就OK了（我是在上一步就完成了，所以这一步是参考网上的）
